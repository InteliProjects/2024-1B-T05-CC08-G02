Glossário - Linguagem FOFI para Terapeutas Ocupacionais

Breve Descrição da Linguagem FOFI

FOFI (Framework Orientado para Facilitação Inclusiva) é uma linguagem de programação projetada para criar atividades interativas e inclusivas, especialmente desenvolvida para ser utilizada por terapeutas ocupacionais em sessões com crianças com Transtorno do Espectro Autista (TEA). A FOFI permite a criação de brincadeiras programáveis que ajudam no desenvolvimento de habilidades cognitivas e motoras das crianças de forma lúdica e envolvente.

Principais Componentes da Linguagem FOFI

Tipos de Dados

  • numero: Representa números inteiros.
  • binario: Representa valores booleanos (verdadeiro ou falso).
  • texto: Representa cadeias de caracteres (strings).

Atribuição

: (dois pontos) é utilizado para atribuir valores a variáveis.

Operadores

  • Aritméticos: `+` (adição), `-` (subtração), `*` (multiplicação), `/` (divisão inteira), `%` (resto da divisão), `^` (exponenciação).
  • Relacionais: `=` (igual), `!=` (diferente), `>` (maior), `<` (menor), `>=` (maior ou igual), `<=` (menor ou igual).
  • Lógicos: `e` (AND), `ou` (OR), `nao` (NOT).

Estruturas de Fluxo

  • Condicionais: `se (condição) { ... } senao { ... }`
  • Repetição: `enquanto (condição) { ... }` e `repita (número de vezes) { ... }`

Marcadores de Bloco

  • { (abre um bloco)
  • } (fecha um bloco)

Terminador de Declarações

; (ponto e vírgula) é usado para finalizar declarações.

Comentários

  • /# ... #/: Comentário de várias linhas.
  • # ...: Comentário de uma única linha.

Sintaxe Básica (Exemplo)

programa
                        "Descrição do programa (como uma string)"
                        
                        /# Este é um comentário de múltiplas linhas #/
                        
                        # Comentário de uma única linha
                        var
                        numero x, y, z; 
                        texto a, b;
                        
                        {
                            x: 0;
                            y: 5;
                            a: "ola";
                            b: "mundo";
                        
                            se (a = b e b != a){
                                mostrar("Ola");
                            }
                            senao {
                                mostrar("nao sei");
                            }
                        
                            enquanto (x < y) {
                                x: x + 1;
                                mostrar(x);
                            }
                        
                            repita(10) {
                                /# o que estiver aqui dentro será repetido 10 vezes. #/
                            }
                        }

Funções Prontas (Built-in)

  • Funções de Entrada de Dados:
    • ler_numero(msg): Solicita um número inteiro do usuário.
    • ler_binario(msg): Solicita uma confirmação (verdadeiro/falso) do usuário.
    • ler(): Espera uma entrada do tapete sensível ao toque.
    • consultar(): Verifica a última entrada do tapete.
  • Funções de Manipulação de Imagens e Figuras:
    • criar_figura(tipo, cor, x, y, tamanho): Cria uma figura (círculo ou quadrado) na tela.
    • criar_imagem(arq, x, y): Carrega uma imagem no canvas.
    • colidiu(ref1, ref2): Verifica se duas figuras ou imagens colidiram.
    • aleatorio(min, max): Gera um número aleatório entre `min` e `max`.
  • Funções de Saída de Dados:
    • mostrar(valor): Exibe um valor na tela.
    • limpar(): Limpa o canvas.
    • inicializar_com_cor(cor): Define a cor de fundo do canvas.
    • inicializar_com_imagem(arq): Define uma imagem de fundo para o canvas.
    • redefinir_figura(ref, tipo, cor, x, y, tamanho): Modifica uma figura existente.
    • redefinir_imagem(ref, arq, x, y): Modifica uma imagem existente.
    • mover(ref, dx, dy): Move uma figura existente.
    • destacar(ref): Destaca uma figura, tornando as outras translúcidas.
    • reverter_destaque(): Reverte o destaque das figuras.
    • tocar(arq): Reproduz um arquivo de áudio.
    • esperar(t): Pausa a execução do programa por `t` milissegundos.

Considerações Finais

A linguagem FOFI foi desenvolvida para ser acessível e intuitiva, permitindo que terapeutas ocupacionais criem atividades interativas que apoiem o desenvolvimento das crianças com TEA. Sua simplicidade e foco em elementos visuais e de interação tornam a FOFI uma ferramenta poderosa para criar experiências educativas e terapêuticas personalizadas.
